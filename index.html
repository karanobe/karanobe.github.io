Flex

Flex was my final project during my attendance at DevBootcamp. This project uses a Ruby on Rails server, Postgres database, and a raw Javascript front-end. Essentially, this web app allows users to meet gym partners efficiently. A Tinder for Gym Buddies, if you will! I built this as a robust solution for any client. All the controllers on the backend return JSON objects, and the front-end converts that data into HTML with design elements. The benefit of the server returning JSON is that the HTML rendering on the front-end is much faster, and the data extraction from the server-side is also more efficient.
